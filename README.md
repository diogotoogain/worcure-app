# WorCure App

## Overview
The WorCure App is designed to help users manage their wellness and improve their daily habits. This application provides tools for tracking various aspects of health and wellness, offering insights and recommendations based on user input.

## Features
- **User-friendly Interface**: Easy navigation and intuitive design.
- **Habit Tracking**: Monitor daily habits and routines.
- **Wellness Insights**: Receive personalized recommendations and insights based on tracked data.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/diogotoogain/worcure-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd worcure-app
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
